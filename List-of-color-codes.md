The color codes are used in event messages to change the color. When of these appear anywhere in the text, the color of the text will be changed. The code itself will be invisible.

| Code | Note |
| ------ | ------ |
| `^BG` | Default color |
| `^TC` | Team color |
| `^F1` | |
| `^F2` | |
| `^F3` | |
| `^F4` | |
| `^1` | Red |
| `^2` | Green |
| `^3` | Yellow |
| `^4` | Dark Blue |
| `^5` | Light Blue |
| `^6` | Pink |
| `^7` | White |
| `^8` | Medium Gray |
| `^9` | Light Gray |
| `^0` | Dark Gray |
